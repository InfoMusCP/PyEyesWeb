Equilibrium

Barycenter and equilibrium analysis for postural snapshots.

This module provides features for analysing postural balance. It computes the Barycenter (Center of Mass) of a set of joints and relates it to specific support nodes (e.g. feet) to estimate stability.

Typical use cases include:

  • Detecting loss of balance in dance or sports.
  • Measuring weight distribution between left and right sides.
  • Estimating postural stability via the relationship between the barycenter and the base of support.

class Equilibrium(StaticFeature):
    """Compute postural equilibrium and barycenter metrics.

    Computes the mean position (barycenter) of all provided joints and, optionally,
    computes the proximity of this barycenter to two specific support joints
    (the "feet").

    Read more in the [User Guide](../../user_guide/theoretical_framework/low_level/postural_balance.md).

    Parameters
    ----------
    support_indices : tuple of (int, int) or None, optional
        Indices of the left and right support nodes (e.g. feet) in the input
        array.  If `None` (default), only the barycenter is computed.

    Examples
    --------
    >>> eq = Equilibrium(support_indices=(10, 11))
    """

    def __init__(self, support_indices: Optional[tuple[int, int]] = None):
        super().__init__()
        self.support_indices = support_indices

    def compute(self, frame_data: np.ndarray) -> EquilibriumResult:
        """Compute the barycenter and support-relative distances.

        Parameters
        ----------
        frame_data : numpy.ndarray of shape (N_signals, N_dims)
            Snapshot of joint positions.

        Returns
        -------
        EquilibriumResult
            Result containing the barycenter vector and, if `support_indices`
            were provided, the distances to support nodes.
        """
        if frame_data.shape[0] == 0:
            return EquilibriumResult(is_valid=False)

        # Barycenter: Mean of all nodes
        barycenter = np.mean(frame_data, axis=0)

        dist_l, dist_r, dist_diff = None, None, None

        if self.support_indices is not None:
            idx_l, idx_r = self.support_indices
            if 0 <= idx_l < frame_data.shape[0] and 0 <= idx_r < frame_data.shape[0]:
                pos_l = frame_data[idx_l]
                pos_r = frame_data[idx_r]

                dist_l = float(np.linalg.norm(barycenter - pos_l))
                dist_r = float(np.linalg.norm(barycenter - pos_r))
                dist_diff = dist_l - dist_r

        return EquilibriumResult(
            barycenter=barycenter,
            dist_l=dist_l,
            dist_r=dist_r,
            dist_diff=dist_diff
        )

@dataclass(slots=True)
class EquilibriumResult(FeatureResult):
    """Output contract for the Equilibrium feature.

    Attributes
    ----------
    barycenter : numpy.ndarray of shape (N_dims,)
        The computed geometric center of the selected joints.
    dist_l : float or None
        Euclidean distance from the barycenter to the left support node.
    dist_r : float or None
        Euclidean distance from the barycenter to the right support node.
    dist_diff : float or None
        Signed difference `dist_l - dist_r`.  Positive values indicate the
        barycenter is closer to the right side.
    """

    barycenter: Optional[np.ndarray] = None
    dist_l: Optional[float] = None
    dist_r: Optional[float] = None
    dist_diff: Optional[float] = None
