Rarity

Rarity analysis of motion samples relative to their distribution.

This module provides a measure of how rare the most recent motion sample is relative to the distribution observed within a sliding window. High rarity indicates that the current movement state is atypical (infrequent) compared to recent history.

The metric combines:

  • The bin distance between the most-probable bin and the bin of the latest sample in a histogram of the window data.
  • The probability difference between those two bins.

Rarity is estimated by building a histogram of all values in the current window and comparing the bin of the latest sample to the most probable bin. The metric captures both how far the current sample is from the peak of the distribution and how unlikely that bin is.

Read more in the User Guide.

class Rarity(DynamicFeature):
    """Compute the rarity of the latest sample relative to the window distribution.

    Rarity is estimated by building a histogram of all values in the current
    window and comparing the bin of the **latest sample** to the **most
    probable bin**.  The metric captures both how far the current sample is from
    the peak of the distribution and how unlikely that bin is.

    Read more in the [User Guide](../../user_guide/theoretical_framework/analysis_primitives/analysis_primitives.md).

    Parameters
    ----------
    alpha : float, optional
        Scaling factor applied to the raw rarity score.  Must be a positive
        finite number.  Defaults to `0.5`.

    Notes
    -----
    The rarity metric is computed as:

    $$
    \\text{rarity} = d_1 \\cdot d_2 \\cdot \\alpha
    $$

    where $d_1$ is the absolute bin index distance between the
    most-probable bin and the bin of the latest sample, and $d_2$ is the
    difference in probability between those two bins.
    """

    def __init__(self, alpha: float = 0.5):
        super().__init__()
        self.alpha = alpha

    @property
    def alpha(self) -> float:
        """Scaling factor for the rarity score."""
        return self._alpha

    @alpha.setter
    def alpha(self, value: float):
        self._alpha = validate_numeric(value, "alpha")

    def compute(self, window_data: np.ndarray) -> RarityResult:
        r"""Compute the rarity score for a given window of motion.

        Rarity is calculated as the negative log-likelihood of the current
        movement state under a Gaussian model.

        Parameters
        ----------
        window_data : numpy.ndarray of shape (Time, N_signals, N_dims)
            All motion samples in the current window.

        Returns
        -------
        RarityResult
            Computed rarity value.  Returns `RarityResult(is_valid=False)`
            if the window contains fewer than 2 samples or if the data
            has zero variance.

        Notes
        -----
        The rarity index follows:

        $$
        \text{rarity} = -\ln(P(\text{state}))
        $$

        where $P(\text{state})$ is the probability density of the current
        window under a Gaussian Distribution.
        """
        # Flatten to 1D array of samples
        samples = window_data.ravel()
        n_samples = len(samples)

        if n_samples < 2:
            return RarityResult(is_valid=False)

        # Number of bins
        n_bins = max(int(np.sqrt(n_samples)), 1)
        counts, bin_edges = np.histogram(samples, bins=n_bins)
        probabilities = counts / n_samples

        # Most probable bin vs Current sample bin
        most_probable_bin_index = np.argmax(probabilities)
        most_probable_p = probabilities[most_probable_bin_index]

        last_sample = samples[-1]
        last_sample_bin_index = np.searchsorted(bin_edges, last_sample, side='right') - 1
        last_sample_bin_index = np.clip(last_sample_bin_index, 0, n_bins - 1)
        last_sample_p = probabilities[last_sample_bin_index]

        d1 = abs(most_probable_bin_index - last_sample_bin_index)
        d2 = most_probable_p - last_sample_p

        rarity = d1 * d2 * self._alpha
        return RarityResult(rarity=float(rarity))
