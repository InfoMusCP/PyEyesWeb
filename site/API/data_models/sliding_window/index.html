Sliding Window

Circular buffer implementation for real-time motion data streaming.

This module provides the SlidingWindow class, a memory-efficient circular buffer designed for high-frequency motion data. It supports dynamic resizing and provides convenient methods for exporting window contents as NumPy tensors or flat arrays.

SlidingWindow

Memory-efficient circular buffer for streaming motion data.

A SlidingWindow stores a fixed-length history of motion frames. As new frames are appended, the oldest ones are automatically discarded. The internal storage uses a pre-allocated NumPy array for maximum performance.

Read more in the User Guide.

Parameters:
  • window_size (int) –

    Maximum number of frames the window can hold. Must be ≥ 1.

class SlidingWindow:
    """Memory-efficient circular buffer for streaming motion data.

    A `SlidingWindow` stores a fixed-length history of motion frames.  As new
    frames are appended, the oldest ones are automatically discarded.  The
    internal storage uses a pre-allocated NumPy array for maximum performance.

    Read more in the [User Guide](../../user_guide/theoretical_framework/theoretical_framework.md).

    Parameters
    ----------
    window_size : int
        Maximum number of frames the window can hold.  Must be ≥ 1.
    """

    def __init__(self, window_size: int):
        self._window_size = window_size
        self._buffer = None
        self._head = 0
        self._count = 0

    @property
    def window_size(self) -> int:
        """Maximum capacity of the sliding window."""
        return self._window_size

    @window_size.setter
    def window_size(self, value: int):
        if value != self._window_size:
            self._resize(value)

    def __len__(self) -> int:
        """Return the number of frames currently in the window."""
        return self._count

    def is_full(self) -> bool:
        """Check if the window has reached its maximum capacity."""
        return self._count == self._window_size

    def append(self, frame_data: np.ndarray):
        """Add a new frame to the end of the window.

        If the window is full, the oldest frame is overwritten.

        Parameters
        ----------
        frame_data : numpy.ndarray
            Movement data for the current frame.  The internal buffer's shape
            is initialized based on the shape of the first frame appended.

        Raises
        ------
        ValueError
            If `frame_data` shape is inconsistent with previously appended
            frames.
        """
        arr = np.asarray(frame_data)

        # Initialize buffer on first append
        if self._buffer is None:
            self._buffer = np.zeros((self._window_size, *arr.shape), dtype=arr.dtype)

        if arr.shape != self._buffer.shape[1:]:
            raise ValueError(
                f"Inconsistent frame shape. Expected {self._buffer.shape[1:]}, got {arr.shape}"
            )

        self._buffer[self._head] = arr
        self._head = (self._head + 1) % self._window_size
        self._count = min(self._count + 1, self._window_size)

    def _resize(self, new_size: int):
        if self._buffer is not None:
            # Preserve current data if we already have a buffer
            current_data, _ = self.to_tensor()
            self._buffer = None  # Force re-initialization
            self._window_size = new_size
            self._head = 0
            self._count = 0
            for frame in current_data[-new_size:]:
                self.append(frame)
        else:
            self._window_size = new_size

    def reset(self):
        """Clear all data from the window and reset counters.

        The internal buffer is kept allocated but logically emptied.
        """
        self._head = 0
        self._count = 0

    def to_tensor(self) -> Tuple[np.ndarray, bool]:
        """Export the window contents as a 3D NumPy tensor.

        Returns
        -------
        tensor : numpy.ndarray of shape (Time, N_signals, N_dims)
            The current window of data in chronological order.
        is_full : bool
            True if the window represents a full `window_size` sequence.
        """
        if self._count == 0:
            return np.array([]), False

        if self._count < self._window_size:
            return self._buffer[:self._count].copy(), False

        # Circular shift to get chronological order
        return np.roll(self._buffer, -self._head, axis=0), True

    def to_flat_array(self) -> np.ndarray:
        """Export the window contents as a flat 1D array.

        Convenient for passing the entire window to features that ignore
        internal spatial structure (e.g. Statistical Moments).

        Returns
        -------
        numpy.ndarray
            1-D flattened array of all samples in chronological order.
        """
        tensor, _ = self.to_tensor()
        return tensor.ravel()
